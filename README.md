#Sorting Visualizer (4-Way Comparison)

An advanced sorting visualizer that compares **4 algorithms simultaneously** with animations and performance tracking.

---

## 🚀 Features

- 🔄 4 algorithms side-by-side
- 🎨 Smooth animations
- 🌈 Color-coded comparisons (compare, swap, sorted)
- ⏱️ Execution time (ms)
- 📊 Big-O complexity display
- 📥 Custom array input
- 🎲 Random array generator

---

## 🧮 Algorithms

- Bubble Sort
- Selection Sort
- Insertion Sort
- Merge Sort
- Quick Sort
- Heap Sort
- Radix Sort

---

## 🛠️ Tech Stack

- HTML
- CSS
- JavaScript

---

## ▶️ How to Run

1. Download project  
2. Open `index.html`  
3. Select algorithms  
4. Enter or generate array  
5. Click Start
